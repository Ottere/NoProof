## pip install
시작하기 앞서 프로그램 실행에 필요한 모듈을 설치하겠습니다.
  
  
  > pip install flask  
  pip install sqlalchemy  
  pip install flask-sqlalchemy  
  pip install selenium  
  pip install pymysql 
  
이로써 필요한 모듈은 모두 설치가 끝났습니다.
